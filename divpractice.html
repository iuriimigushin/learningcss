DIV vs OBJECT

    Div tag provide actions within blocks while Object tag provide actions within objects.
    Main usage of Object tag was Flash. Object tag is rarely use nowadays.
    Object is something that contain some elements, so it should be widely than div.

    Attributes:
  • Width of object,
  • Height of object,
  • Link to it's object (it's source, data).

IMG tag for images

    Img one-side tag for images.
    Src - source,
    Alt - error message if source is unavailable. Usually use Name of image.
    Title - name of image. Same as Alt usually.
    Width & Height. No values mean original size. Both size force resize. One size - force proportional resize.
