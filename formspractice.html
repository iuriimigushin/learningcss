Tag Input

Forms are visual objects.
      Form is bundle of different fields.
      FORM - is main tag.
      INPUT - one-side tag for anykind input.
      NAME (atr.) - unique name of element.
      ID (atr.) - unique id of element.
      TYPE (atr.) - type of content:
        - "text"
        - "hidden"
        - "submit"
        - "password"
        - "number"
        - "email"
        - "checkbox"
        - "radio"
        - "color"
        - "date"
        - "time"
        - "file"
        - "range"
      VALUE (atr.) - default value of element. For submit type change default button text.
      Default value for TEXTAREA should be located BETWEEN parantheses.
      PLACEHOLDER (atr.) - in-field tooltip. Hiden if field have default value but will appear as soon as field become empty.
      LABEL - text message. For example text name for input field.
      LABEL new attribute:
      FOR (atr.) - make link between input field and label. Activate input field by clicking on label. Write id of corresponding input element.
      FOR new attirubte:
      ACTION (atr.) - indicates the path to the handler file.
      METHOD - transit data by public or hidden:
        - "get" - public. User will see it as request in browser address field.
          For example:
          .../formpractice.html
          .../formpractice.html?firstname=Myname&?lastname=Mylastname
        - "post" - hidden.
      TABINDEX (atr.) - manual option provide switching between elements by TAB key.
