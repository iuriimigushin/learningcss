Tag TABLE uses only for tables. Do not use it for whole page layout. Table contain lines and columns in lines. TABLE - table.
TR - line.
TD - generate cell in line with regular and left-aligned text (to make common cells in columns).
TH - generate cell in line with bold and centered text (to make headers of columns for example).
BORDER - attribute of border.
- header in table's elements.
TITLE - tooltip with name of table.
