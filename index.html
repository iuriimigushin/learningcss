Connecting CSS to HTML

    Connecting ways:

  • External CSS (link)
  • External CSS (style)
  • Internal CSS
  • Inline CSS

External CSS (link)

Example HTML:
<head>
<link href="style.css" rel="stylesheet">
<head>
 Example CSS:
body {
color:blue;
}

External CSS (style)

Example HTML:
<style>
@import "style.css";
<style >
 Example CSS:
body {
color:blue;
}
 This way is rare but useful in another purpose.
Similiar way provide complex multi CSS-files structure: CSS-files import into main CSS file.
Example HTML:
<head>
<style>
@import "style.css";
</style>
<head>
 Example CSS:
@import "fonts.css"
@import "colors.css"

body {
color:blue;
}
