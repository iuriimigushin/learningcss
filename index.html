Connecting CSS to HTML

    Connecting ways:

  • External CSS (link)
  • External CSS (style)
  • External multiply CSS (style)
  • Internal CSS
  • Inline CSS
  • Why it is useful to use an external CSS file

External CSS (link)

This is the regular way.
Example HTML:
<head>
<link href="style.css" rel="stylesheet">
<head>
 Example CSS:
body {
color:blue;
}

External CSS (style)

This way is rare but useful in another purpose.
Example HTML:
<style>
@import "style.css";
<style >
 Example CSS:
body {
color:blue;
}

External multiply CSS (style)

Similiar way provide complex multi CSS-files structure: CSS-files import into main CSS file.
Example HTML:
<head>
<style>
@import "style.css";
</style>
<head>
 Example CSS:
@import "fonts.css"
@import "colors.css"

body {
color:blue;
}

Internal CSS

Used for 1-2 styles while common CSS have much more styles.
Example HTML:
<head>
<style>
body {
color:blue;
}
</style>
<head>

Inline CSS

For quick cases.
Example HTML:
<body style="color:lightblue";>

Why it is useful to use an external CSS file

Any far partitions of website will have same style.
It's possible to use same style at any website partition.
