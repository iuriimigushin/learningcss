Lists

HTML have 3 types of lists.

    Ordered list:
  1. list item 1
  2. list item 2
  3. list item 3
    Unordered list:
  • list item 1
  • list item 2
  • list item 3

Description list:
Name 1
Value 1
Name 2
Value 2
Name 3
Value 3

HTML supports lists mix

    Ordered list
  1. List item 1
  2. List item 2
    1. List item 2.1
    2. List item 2.2
    3. List item 2.3
  3. List item 3

    Ordered list
  1. List item 1
  2. List item 2
    • List item 2.A
    • List item 2.B
    • List item 2.C
  3. List item 3

    Unordered list with multiply levels:
  • level 1 list item 1
  • level 1 list item 2
      • Unordered list with multiply levels:
    • level 2 list item 1
    • level 2 list item 2
        • Unordered list with multiply levels:
      • level 3 list item 1
      • level 3 list item 2
      • level 3 list item 3
    • level 2 list item 3
  • level 1 list item 3
